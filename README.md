# Side Quest Week 9

## Setup and Interaction Instructions

**Running the Sketch**

Right-click on index.html and open in broswer using Live Server.

**Controls:**

Press **Higher ▲** or **Lower ▼** to guess. Reach the target score shown in the HUD to advance to the next level. Equal cards count as wrong — you must commit to a direction.

**Debug shortcuts:**

- **D** — toggle debug panel
- **1 / 2 / 3** — jump directly to that level
- **S** — jump to start screen
- **W** — jump to win screen

## Assets

No external assets used. All visuals are generated with p5.js.
